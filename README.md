# EcoFact Bot 🌍                                   -                       Mikhail Rochev


Telegram-бот, который делится интересными фактами о глобальном потеплении и экологии.

## Описание

Образовательный бот, созданный для повышения осведомленности о проблемах изменения климата. При запуске бот приветствует пользователя и предлагает получить случайный факт о глобальном потеплении.

## Функциональность

- **/start** — приветственное сообщение с инструкцией по использованию
- **/fact** — получение случайного факта о глобальном потеплении
- **/solution** — получение случайного совета о том, как помочь планете
- **/challenges** — получение случайного эко-челленджа на сегодня
- **/done** — отметить выполнение эко-челленджа и получить баллы
- **/mystats** — просмотр личной статистики (факты, решения, выполненные челленджи)

## Технологии

- Python 3.x
- Библиотека [pyTelegramBotAPI] (telebot)

## Заключение
  EcoFact Bot — это не просто сборник интересных фактов, а ваш персональный проводник в мир экологичного образа жизни. В современном мире проблема глобального потепления становится всё более острой, и мой проект нацелен на то, чтобы превратить сложную научную информацию в простые, понятные и запоминающиеся сообщения.
